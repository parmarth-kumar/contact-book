# Contact Book

A console-based Contact Book application written in C++.

## Features

* Add new contacts
* View all contacts
* Search contacts by name
* Update contact phone numbers
* Delete contacts
* Handle duplicate contact names
* Validate phone numbers (exactly 10 digits)
* Validate menu input

## Concepts Practiced

* Arrays
* Functions
* Loops
* Conditional Statements
* Input Validation
* Searching Algorithms
* Data Updating
* Data Deletion
* Code Reusability
* Passing Arrays to Functions

## How It Works

The program stores contact names and phone numbers using parallel arrays.

Users can:

1. Add contacts
2. View all contacts
3. Search contacts
4. Update contacts
5. Delete contacts
6. Exit the program

When multiple contacts have the same name, the user can choose which contact to update or delete.
